# Prompt Expander

A Python application that transforms short, simple prompts into detailed, high-quality prompts for generative AI tools. This project helps users improve prompt quality for image generation, text generation, and other AI workflows.

## Features

- Expand short prompts into richer, more descriptive prompts
- Save expanded prompts to files for future use
- Batch process multiple prompts at once
- Organize outputs automatically
- Easily swap between different LLM providers or models
- Command-line interface for quick usage

A futuristic city
